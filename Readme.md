### SETUP:
```git clone https://github.com/suvanbalu/Indic_Language_Support.git ```

#### Frontend:
- Move to the client directory: ```cd frontend```
- Install the required libraries and packages: ```npm install```
- To start the client: ```npm start```

#### Backend:
- Install flask:  ``` pip install flask ```
- Move to the Notebooks directory: ``` cd Notebooks ```
- Execute each cell in the ``` initialization_backend.ipynb ``` jupyter notebook.
- Execute the main.py file: ``` python main.py ```
